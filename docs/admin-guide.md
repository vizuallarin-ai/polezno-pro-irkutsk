## 1. Вход в систему

### Шаг 1.1. Откройте админку

1. В браузере перейдите по адресу: **https://irkportal.ru/admin**
2. Дождитесь загрузки страницы входа.

### Шаг 1.2. Войдите под своим аккаунтом

1. В поле **Email** введите ваш email администратора.
2. В поле **Password** введите пароль.
3. Нажмите кнопку **Login** (или **Войти**).
4. Если данные верны — откроется **главная панель (Dashboard)**.

> **Если забыли пароль** — обратитесь к разработчику: на сервере выполняется команда `npm run reset-admin-password`.

### Шаг 1.3. Язык интерфейса

Интерфейс CMS по умолчанию на **русском**. Переключатель языка (если виден) — в правом верхнем углу или в меню профиля.

## 2. Обзор интерфейса

После входа вы видите:

| Зона экрана | Что там |
|-------------|---------|
| **Левое меню** | Коллекции (Маршруты, Заявки, Материалы…) и глобальные настройки |
| **Верхняя панель** | Поиск, профиль, выход |
| **Центр (Dashboard)** | Сводка: счётчики маршрутов, заявок, фото на модерации |
| **Быстрые действия** | Кнопки «+ Маршрут», «+ Статья», «Настройки» и др. |

### Полезные приёмы

- **Клик по карточке счётчика** на Dashboard — сразу открывает нужный список (например, «Новые заявки»).
- Кнопка **«Настройки»** в быстрых действиях ведёт в глобальные настройки сайта.
- После сохранения записи сайт обновляется автоматически (обычно в течение минуты).

## 4. Работа с медиафайлами

Изображения для настроек, маршрутов и статей хранятся в коллекции **Медиа**.

### Загрузить новый файл

1. В левом меню нажмите **Медиа**.
2. Нажмите **Create New** (или **Создать** / **+**).
3. Перетащите файл в зону загрузки **или** нажмите **Select a file** и выберите файл с компьютера.
4. При необходимости заполните **Alt** (описание для доступности).
5. Нажмите **Save** (внизу или вверху справа).

**Поддерживаемые форматы:** JPEG, PNG, WebP, SVG, MP4, WebM, MP3, PDF.  
**Максимальный размер:** 50 МБ.

### Выбрать уже загруженный файл

В любом поле типа «загрузка» (например, **Фото автора**):

1. Нажмите на поле загрузки.
2. Выберите вкладку **Choose from existing** (или **Выбрать из существующих**).
3. Кликните по нужному файлу.
4. Подтвердите выбор.

## 5. Статусы публикации

Большинство материалов имеют поле **Статус** (в боковой панели справа или в основной форме):

| Статус | Значение |
|--------|----------|
| **Черновик (draft)** | Не виден на сайте |
| **Опубликован (published)** | Виден посетителям |

### Как опубликовать

1. Откройте запись (маршрут, статью, товар…).
2. Заполните обязательные поля (отмечены звёздочкой *).
3. В поле **Статус** выберите **Опубликован**.
4. Нажмите **Save**.

### Фото — дополнительная модерация

Для фотоархива нужны **одновременно**:

- **Статус публикации** = Опубликован
- **Модерация** = Одобрено
- **Тип прав** ≠ «Неизвестно»
- Галочка **Подтверждение прав** включена

Без этого фото не появится на `/explore/photos`.

## 6. Заявки с сайта

Все формы сайта (контакты, бизнес, сувениры, AR) попадают в **Заявки**.

### Просмотр новых заявок

1. На Dashboard нажмите карточку **«Новые заявки»**  
   **или** в меню выберите **Заявки**.
2. В списке отсортируйте по дате (новые сверху).
3. Кликните по строке — откроется карточка заявки.

### Обработка заявки

1. Прочитайте поля: имя, контакт, сообщение, источник (`source`), URL страницы.
2. Свяжитесь с клиентом вне CMS.
3. Измените **Статус** заявки (например: `new` → `in_progress` → `closed`).
4. Нажмите **Save**.

> Пока не настроен email (Resend), заявки **не приходят на почту** — проверяйте админку регулярно.

## 7. Этап 0 — пошаговая настройка

**Цель этапа 0:** реальные контакты, живная главная, работающие уведомления о заявках, подтверждение в поисковых системах.

**Время:** 1–2 часа в админке + настройка почты с разработчиком.

---

### Блок 0.1. Контакты и мессенджеры

**Путь:** Dashboard → **Настройки** (быстрое действие)  
**или:** левое меню → **Globals** → **Настройки сайта**  
**URL:** `/admin/globals/site-settings`

#### Шаги

1. Откройте **Настройки сайта**.
2. Прокрутите до группы полей **Контакты**.
3. Заполните поля:

   | Поле в админке | Что ввести | Пример |
   |----------------|------------|--------|
   | **Телефон** | Реальный номер | `+7 (914) 123-45-67` |
   | **Email** | Рабочая почта | `info@example.ru` |
   | **Telegram** | Полная ссылка | `https://t.me/your_channel` |
   | **MAX** | Ссылка на чат MAX | если используете |
   | **WhatsApp** | Ссылка `wa.me/...` | по желанию |
   | **ВКонтакте** | Ссылка на группу/страницу | по желанию |
   | **YouTube** | Ссылка на канал | по желанию |
   | **Instagram** | Ссылка на профиль | по желанию |

4. Прокрутите до группы **Заявки и согласия**.
5. В поле **Основной мессенджер** выберите: **Telegram**, **MAX** или **Email** — что показывать первым на сайте.
6. В поле **Email для уведомлений о заявках** укажите почту, куда хотите получать письма.
7. Убедитесь, что галочка **Email-уведомления о новых заявках** включена.
8. Нажмите **Save** (вверху справа).

#### Проверка

1. Откройте в новой вкладке: https://irkportal.ru/contact
2. Убедитесь, что отображаются ваши Telegram / email / телефон.
3. Телефон-заглушка `000-00-00` на сайте **не показывается** — это нормально, если поле пустое или заглушка.

### Блок 0.3. Главная страница — Hero и автор

**Тот же раздел:** Настройки сайта.

#### Hero (первый экран)

1. Найдите поля группы Hero:
   - **Hero — бейдж** (маленькая плашка сверху)
   - **Hero — заголовок**
   - **Hero — подзаголовок**
2. Введите финальные тексты.
3. *(Опционально)* **Hero-видео** — загрузите WebM или выберите из Медиа.
4. *(Опционально)* **Постер Hero-видео** — картинка до воспроизведения видео.

#### Блок автора

1. **Имя автора** — например: `Алёна Ямщикова`.
2. **Роль автора** — например: `Автор навигатора и гид`.
3. **Короткий текст об авторе** — 2–3 предложения для главной и страницы «О проекте».
4. **Фото автора:**
   - Кликните поле **Фото автора**.
   - **Create New** — загрузите портрет **или** **Choose from existing** — выберите из Медиа.
5. Нажмите **Save**.

#### Кнопки на главной (CTA)

1. Группа **Главный CTA:**
   - **Текст кнопки** — например: `Спланировать поездку`
   - **Ссылка** — `/business`
2. Группа **Вторичный CTA:**
   - **Текст кнопки** — например: `Маршруты`
   - **Ссылка** — `/map`
3. **Save**.

#### Проверка

Откройте https://irkportal.ru/ — обновите страницу (Ctrl+F5). Проверьте заголовок, фото автора и кнопки.

### Блок 0.4. SEO главной и соцсети

**Тот же раздел:** Настройки сайта.

### Блок 0.6. Email-уведомления о заявках

**Часть в админке (вы):**

1. Настройки сайта → **Email для уведомлений о заявках** — ваш email.
2. Галочка **Email-уведомления** — включена.
3. **Save**.

**Часть с разработчиком:**

1. Создайте аккаунт на [resend.com](https://resend.com).
2. Подтвердите домен отправки (или используйте тестовый режим).
3. Передайте разработчику ключ **RESEND_API_KEY**.
4. Разработчик добавит на сервер переменные: `RESEND_API_KEY`, `EMAIL_FROM`, `EMAIL_TO`.

#### Тест заявки

1. Откройте https://irkportal.ru/business
2. Заполните форму тестовыми данными.
3. Отправьте.
4. Проверьте:
   - **Заявки** в админке — новая запись со статусом `new`
   - почту (после настройки Resend)

## 8. Частые вопросы

### Сохранил, но на сайте не видно

1. Проверьте **Статус** = **Опубликован**.
2. Подождите 1–2 минуты и обновите страницу с Ctrl+F5.
3. Для фото — проверьте модерацию и права.

### Не могу опубликовать фото

CMS блокирует публикацию без:
- модерации «Одобрено»;
- указанного типа прав (не «Неизвестно»);
- галочки подтверждения прав.

### Где изменить меню сайта?

Пункты меню задаются в коде. Коллекция **Навигация** пока скрыта. Для изменения меню — запрос к разработчику.

### Как добавить второго администратора?

1. Меню → **Пользователи** → **Create New**.
2. Email, пароль, роль **admin**.
3. **Save**.

### Как выйти?

Правый верхний угол → иконка профиля → **Log out** / **Выйти**.
